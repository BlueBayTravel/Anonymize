# Anonymize

Anonymize IPv4 and IPv6 addresses.

# Installation

This version requires [PHP](https://php.net/) 7.1 or 7.2.

To get the latest version, require the project using [Composer](https://getcomposer.org/):

```
$ composer require bluebaytravel/anonymize
```
